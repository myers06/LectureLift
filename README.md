#  LectureLift

LectureLift is an AI-powered learning assistant that helps students summarize lectures, extract notes from study materials, and generate quizzes using AI.

## Features
-  Record lectures and get AI summaries  
-  Upload PPTs/notes to get key points  
-  Generate quizzes from study material  
-  Store files 

## Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Python (Flask/Streamlit)  
- AI: Google Gemini AI  
- Database: JSON-based storage

## How to Run
1. Clone the repository  
2. Install dependencies  
3. Run the application
